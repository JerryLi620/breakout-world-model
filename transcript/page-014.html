# Claude Code Transcripts Tool

## What It Is

This tool converts Claude Code session files into clean, mobile-friendly HTML transcripts with pagination. As stated in the README: "Convert Claude Code session files (JSON or JSONL) to clean, mobile-friendly HTML pages with pagination."

## Installation

Install using `uv`:
```
uv tool install claude-code-transcripts
```

Or run without installing:
```
uvx claude-code-transcripts --help
```

## Extracting a Recent Session

The quickest way to view a recent local session is simply:
```
claude-code-transcripts
```

This command displays an interactive picker to select a session from your local Claude Code projects stored in `~/.claude/projects`, generates HTML, and opens it in your default browser automatically.

The tool offers four main commands: `local` (default for interactive selection), `web` (for sessions via the Claude API), `json` (for specific files), and `all` (to convert all sessions into a browsable archive).
